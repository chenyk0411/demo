1.1、rabbitMq

-     打开管理界面命令：rabbitmq-plugins enable rabbitmq_management
-     停止：net stop RabbitMQ
-     启动：net start RabbitMQ
-     后台管理地址：http://127.0.0.1:15672/
-     使用默认账号登录：guest/ guest
